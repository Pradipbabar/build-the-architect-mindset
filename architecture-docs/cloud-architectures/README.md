# Cloud Architectures

This directory contains cloud-native architecture designs, patterns, and implementations.

## Structure

- `aws/` - AWS-specific architectures and patterns
- `azure/` - Azure-specific architectures and patterns
- `gcp/` - Google Cloud Platform architectures and patterns
- `multi-cloud/` - Multi-cloud and hybrid cloud designs
- `iac/` - Infrastructure as Code templates and patterns

## Examples

- Serverless architectures (Lambda, Cloud Functions)
- Container orchestration (ECS, EKS, AKS, GKE)
- Data lake and analytics architectures
- High-availability and disaster recovery designs
- Cost-optimized architectures
- Security-focused cloud designs

## Templates

Use these architectures as starting points for your cloud designs. Each design should include:

- Architecture diagram
- Service descriptions
- Security considerations
- Cost estimates
- Deployment instructions
- Monitoring and observability setup
